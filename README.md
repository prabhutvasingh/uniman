# Uniman
A universal package manager for every distro, acting like a Windows `.exe` installer but designed with Linux beginners in mind. Many users switching from Windows carry over the habit of avoiding built-in app stores — the same mindset that kept them away from the Microsoft Store — and instead search for `.exe` installers from official developer websites. Uniman solves that friction by providing a familiar, guided installation experience on Linux, without needing to touch the terminal.

**Features:**
- Better UI compared to the old-school Windows installer experience
- Live, updated system information during installation
- Custom backend and frontend with support for multiple distros including Arch Linux, Ubuntu, Fedora, FreeBSD, and Debian'

**Usage:**
Uniman can be used by runnign the following syntax in the terminal:
`
python3 uniman.py [PACKAGE-NAME]
`
